# Library Review Rules

This directory contains review modules for language/runtime surfaces, third-party libraries, and shared tooling dependencies used by Logseq.

Use these files as focused checklists. A review should load every library file that matches the diff, not just the most obvious one.

## Current modules

- [`clojure-cljs.md`](./clojure-cljs.md) — Clojure/CLJS idioms and JS interop
- [`malli.md`](./malli.md) — schemas, validation, coercion, explain data
- [`promesa.md`](./promesa.md) — promises and async composition
- [`datascript.md`](./datascript.md) — DB snapshots, queries, transactions
- [`missionary.md`](./missionary.md) — tasks, flows, signals, cancellation
- [`rum-react.md`](./rum-react.md) — component lifecycle and render behavior
- [`cljs-time.md`](./cljs-time.md) — date formatter construction and date parsing/formatting
- [`glogi.md`](./glogi.md) — application logging conventions
- [`babashka-cli.md`](./babashka-cli.md) — command-line parsing and scripts
- [`shadow-cljs-node.md`](./shadow-cljs-node.md) — build targets, npm interop, Electron/Node loading
